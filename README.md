## 概要

AozoraTxtViewer は M5Cardputer で青空文庫のテキストを読むためのビューワーです。  
SDカードに UTF-8 形式の `.txt` ファイルを入れるだけで使用できます。

**主な特徴：**
- 青空文庫のルビ記法（`《》`）を自動除去して表示
- TTF日本語フォント対応（サイズ 10/12/14/16px）
- 内蔵フォント対応（12px 固定、SDカード不要）
- ライトモード / ダークモード切替
- ページ・フォント設定の自動保存（しおり：ページ切替ごとに自動保存）
- 節電モード（操作なし2分で減光、5分で消灯）

## SDカードの準備

```
SDカード（ルート直下）
├── texts/
│   ├── 吾輩は猫である.txt    ← UTF-8変換済みテキスト
│   ├── 坊ちゃん.txt
│   └── ...
└── fonts/
    └── NotoSansJP.ttf        ← 日本語TTFフォント（任意）
```

### テキストファイルについて
- エンコーディング：**UTF-8**（必須）
- 改行コード：LF / CRLF どちらも対応
- 青空文庫の公式テキストは Shift-JIS のことが多いため、  
  事前に UTF-8 に変換してから `/texts/` フォルダに入れてください

### フォントファイルについて
- `/fonts/` フォルダに日本語 TTF ファイルを置くと TTF モードが使用可能になります
- フォントがない場合は内蔵フォント（12px 固定）で動作します
- ファイル名は何でも構いません（複数ある場合はアルファベット順で最初のものが読まれます）

## 起動と基本操作

電源を入れると自動的に `/texts/` フォルダのファイル一覧が表示されます。

起動時の動作：
1. SDカードをマウント
2. `/fonts/` から TTF フォントを読み込み（あれば）
3. `/texts/` からテキストファイル一覧を取得
4. テキスト選択画面を表示

## テキスト選択画面

```
┌──────────────────────────┐
│ テキスト選択              │  ← ヘッダー
├──────────────────────────┤
│   吾輩は猫である          │
│ > 坊ちゃん               │  ← 選択中（反転表示）
│   こころ                 │
│   ...                    │
├──────────────────────────┤
│ ;/. ↑↓    Enter 開く     │  ← フッター
└──────────────────────────┘
```

| キー | 動作 |
|------|------|
| `;` | カーソルを上へ |
| `.` | カーソルを下へ |
| `,` | ページアップ（一覧を上にスクロール） |
| `/` | ページダウン（一覧を下にスクロール） |
| `Enter` | 選択したファイルを開く |

ファイルを開くと自動的にページインデックスを構築します（長いファイルは数秒かかります）。  
以前に開いたことがあるファイルは、最後に読んでいたページから再開します。

## キー一覧

### 全画面共通

| キー | 動作 |
|------|------|
| `H` | ヘルプポップアップを表示 |
| `C` | ライト / ダーク モード切替 |
| `L` | 節電モード ON/OFF |
| `-` | バックライトを暗くする |
| `=` | バックライトを明るくする |
| `Q` | TTFフォント / 内蔵フォント 切替（TTF読込済みの場合のみ） |
| `F` | TTFフォントサイズ変更 10→12→14→16px（TTF使用時のみ） |

### 読書画面のみ

| キー | 動作 |
|------|------|
| `.` | 次のページへ |
| `;` | 前のページへ |
| `J` | ページ番号を入力してジャンプ |
| `M` | 青空文庫モード / 通常モード 切替 |
| `V` | フッター表示 ON/OFF |
| `Esc` / `` ` `` / `~` | テキスト選択画面へ戻る |

## フォント設定

### TTFフォント（SDカードに TTF が必要）

`Q` キーで TTF モードに切替後、`F` キーでサイズを変えられます。

| サイズ | 1行の文字数 | 1ページの行数（フッターあり） |
|--------|------------|-------------------------------|
| 10px | 19文字 | 6行 |
| 12px | 19文字 | 6行 |
| 14px | 16文字 | 5行 |
| 16px | 14文字 | 5行 |

フォントサイズを変更すると自動的にページインデックスが再構築されます。  
現在読んでいた位置は保持されます。

### 内蔵フォント（SDカード不要）

`Q` キーで内蔵フォントモードに切替できます（TTF読込済みの場合のみ切替可）。

| サイズ | 1行の文字数 | 備考 |
|--------|------------|------|
| 12px 固定 | 19文字 | SD不要・高速起動 |

## 表示設定

### ライト / ダークモード（`C` キー）

| モード | 背景 | 文字 |
|--------|------|------|
| ライト | 白 | 黒 |
| ダーク | 濃い紺 | 白 |

設定は NVS に保存され、次回起動時も維持されます。  
テキスト選択画面は常にライトモードで表示されます（設定の変更はありません）。

### フッター表示 ON/OFF（`V` キー・読書中のみ）

フッターを非表示にすると画面の表示行数が増えます。  
変更時はページインデックスが自動的に再構築されます。

## 節電・バックライト

### 手動明るさ調整

`-` / `=` キーで5段階（消灯 / 25% / 50% / 75% / 100%）に調整できます。  
現在の明るさレベルはヘッダー右側の数字で確認できます（1〜4）。

### 節電モード（`L` キー）

ヘッダーに `*` が表示されているとき節電モードが ON です。

| 操作なし経過時間 | 動作 |
|-----------------|------|
| 2分 | バックライトを設定値の50%に減光 |
| 5分 | バックライトを消灯（完全消灯） |

キーを押すと即座に通常の明るさに戻ります。  
節電モード OFF（`O` 表示）の場合は常に設定明るさで点灯します。

## しおり機能

読書中に**ページを切り替えるたびに**（`.`/`;` キー）、以下の情報が自動的に保存されます。  
`Esc` キーでテキスト選択画面に戻るときも保存されます。

| 保存される情報 | 説明 |
|--------------|------|
| ページ番号 | 現在読んでいるページ |
| 読書モード | 青空文庫モード / 通常モード |
| フォントモード | TTF / 内蔵フォント |
| TTFフォントサイズ | 10 / 12 / 14 / 16px |

次回同じファイルを開いたとき、保存されたページ・フォント設定で再開します。  
突然のバッテリー切れでも、最後に読んでいたページから再開できます。

- しおりはファイル名ごとに独立して保存されます
- 保存先：M5Cardputerの内部フラッシュ（NVS）
- SDカードを差し替えても保存データは残ります
- SDカードがなく TTF が使えない場合、フォントは内蔵フォントで代替されます

---

## 青空文庫モード / 通常モード

`M` キーで切替できます（読書中のみ）。

### 青空文庫モード（デフォルト）

青空文庫形式のマークアップを自動で処理します：

| マークアップ | 処理 |
|------------|------|
| `漢字《よみ》` | ルビ（読みがな）を除去、漢字のみ表示 |
| `｜漢字《よみ》` | 同上（縦棒付き記法にも対応） |
| `［＃注釈内容］` | 入力注記を除去 |
| `-------` で囲まれた範囲 | 奥付・底本情報をスキップ |

### 通常モード

マークアップを処理せず、ファイルの内容をそのまま表示します。  
青空文庫以外の UTF-8 テキストファイルを読む場合に使用します。

## 設定の保存について

以下の設定は NVS（内蔵フラッシュ）に自動保存されます。

### グローバル設定（全ファイル共通）

| 設定項目 | キー | 保存タイミング |
|---------|------|---------------|
| ライト/ダークモード | `C` | 変更時 |
| フッター表示 | `V` | 変更時 |
| 節電モード | `L` | 変更時 |
| バックライトレベル | `-`/`=` | 変更時 |
| TTF/内蔵フォント（初期値）| `Q` | 変更時 |
| TTFフォントサイズ（初期値）| `F` | 変更時 |

### しおり（ファイルごとに独立）

| 保存される情報 | キー | 保存タイミング |
|--------------|------|---------------|
| ページ番号 | `.`/`;` | ページ切替ごと（自動） |
| 読書モード | `M` | ページ切替時・`Esc` 時 |
| フォントモード | `Q` | ページ切替時・`Esc` 時 |
| TTFフォントサイズ | `F` | ページ切替時・`Esc` 時 |

## トラブルシューティング

**テキストが文字化けする**  
→ テキストファイルが UTF-8 でない可能性があります。テキストエディタや変換ツールで UTF-8 に変換してください。

**フォントが切り替わらない（`Q` キーが効かない）**  
→ SDカードの `/fonts/` フォルダに TTF ファイルがない場合は内蔵フォントのみで動作します。TTF を追加してから再起動してください。

**「/texts/にtxtなし」と表示される**  
→ SDカードの `/texts/` フォルダに `.txt` ファイルがありません。ファイルを追加してリセットしてください。

**ページ数が 1 のまま変わらない**  
→ ファイルが 1 ページに収まるほど短いか、または読み込みに失敗している可能性があります。
